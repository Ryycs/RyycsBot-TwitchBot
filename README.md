# TwitchBot RyycsBot:

Commands for TwitchBot [RyycsBot](https://www.twitch.tv/ryycsbot)

* Check out [this project](https://github.com/Ryycs/BotCommands) to get a  similar bot.
* All Commands have a delay of 5 seconds.
